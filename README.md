# parcoursupTool

## Purpose

Retrieve [Parcoursup](https://www.parcoursup.fr/) offers for a given student.

## Prerequisites

The following programs must be installed:

- Python 3
- Pip

## Usage

### Install dependencies

```bash
$ pip install -r requirements.txt
```

To update them, run:

```bash
$ pip install --upgrade -r requirements.txt
```

### Run

```bash
$ python main.py
```
